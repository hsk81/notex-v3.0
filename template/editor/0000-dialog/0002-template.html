Select Document Template

Maximum Columns of 1

A template which will render the content with at maximum 1 column, independent of the screen available. For print, also number of columns will also be always 1.

Maximum Columns of 2

A template which will render the content with at maximum 2 columns, depending on the screen width available. For print, the number of columns will always be 2.

Maximum Columns of 3

A template which will render the content with at maximum 3 columns, depending on the screen width available. For print, the number of columns will always be 3.

Empty Template

A template, which is bare minimum without any predefined styling at all.

Select Document Template

Single Column

A template which will render the content with a maximum of 1 column, independent of the device screen. For print, the number of columns will always be 1.

Max. Double Column

A template which will render the content with a maximum of 2 columns, depending on the device screen. For print, the number of columns will always be 2.

Max. Triple Column

A template which will render the content with a maximum of 3 columns, depending on the device screen. For print, the number of columns will always be 3.

Empty Template

A template, which is bare minimum without any predefined styling at all.
